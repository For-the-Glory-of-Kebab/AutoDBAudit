# Configuration Folder

> **⚠️ NOTE**: Actual config files are in `.gitignore` and will NOT sync to Git.

## Getting Started

Copy the example files and customize:
```bash
cp audit_config.example.json audit_config.json
cp sql_targets.example.json sql_targets.json
```

## Files

| File | Purpose | Git Status |
|------|---------|------------|
| `audit_config.json` | Audit settings, org info, expected builds | **Ignored** |
| `sql_targets.json` | SQL Server connection targets | **Ignored** |
| `*.example.json` | Templates with all options documented | Committed |
