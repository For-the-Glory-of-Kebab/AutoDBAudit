# AutoDBAudit Resources Directory

This directory is for extensible resources that support the audit workflow.

## PowerShell Scripts

The `powershell/` subdirectory can contain helper scripts for:
- Custom remediation pre/post hooks
- Environment preparation
- Bulk server discovery
- Report post-processing

## Adding Custom Resources

Place any custom scripts, templates, or helper files here. They will be
preserved across tool updates.

## Example Resources

- **Pre-audit checklist templates**
- **Custom SQL query libraries**
- **Organization-specific remediation scripts**
- **Reporting templates**
